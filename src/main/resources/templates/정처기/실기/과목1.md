#### 소프트웨어 아키텍쳐 4+1 뷰

* 요소\

| 유스케이스 뷰 | 다른 뷰 검증 사용(사용자, 개발자 관점)    |
|---------|----------------------------|
| 논리 뷰    | 설계자 관점 (UML)               |
| 프로세스 뷰  | 시스템 비기능적 속성으로 효율적 사용 꽤함    |
| 구현 뷰    | 실제 구현할 수 있는지 확인            |
| 배포 뷰    | 배치 연결 작업 어떻게 실행 되는지 보여주는 뷰 |
